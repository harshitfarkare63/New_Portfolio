# Harshit Farkare — Computer Science (AI) Portfolio

A modern, high-performance, single-page professional portfolio website demonstrating my expertise as an Artificial Intelligence developer, full-stack engineer, and computer science undergraduate. 

![Portfolio Preview Showcase](https://img.shields.io/badge/Status-Active-success) ![Tech Stack](https://img.shields.io/badge/Built_With-HTML_|_CSS-f5a623)

## 🌌 Overview

Designed with an ultra-premium, dark-themed aesthetic featuring **glassmorphism**, dynamic glowing accents, and smooth scroll animations. This portfolio showcases my best work at the intersection of Artificial Intelligence, workflow automation, and scalable software architecture.

## 🚀 Featured Projects Highlights

* **LeadForge AI:** A full-stack AI lead generation SaaS that automates finding prospects, extracting verified contacts, and generating hyper-personalized cold outreach via Machine Learning models. Built using React 19, FastAPI, Python, and Next.js.
* **AI Voice Coding Assistant:** A multi-modal voice-enabled AI chatbot integrated with GitHub. It detects code errors, provides real-time intelligent fixes via the Gemini API, and allows hands-free voice interaction (TTS/STT).
* **AutoSync Platform:** An AI workflow automation engine bridging Excel, CRM, and invoicing systems. Users dictate workflows in natural language while Celery & Redis handle long-running background asynchronous data sync tasks.

## 🛠 Features

* **Zero-Dependency Architecture:** Built completely with pure HTML5 and vanilla CSS3 to guarantee lightning-fast load times.
* **Responsive Design:** Fluidly scales from high-resolution, wide desktop views all the way down to mobile devices seamlessly.
* **Intersection Observer Animations:** Content elegantly fades and glides into view as you scroll down the page, enhancing engagement.
* **Granular Technical Skills Listing:** Data Structures, Algorithms, OOP, Python, Java, Django, Streamlit, React, and GCP configurations.

## 💻 Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** Custom CSS3 (Vars, Flexbox, CSS Grid)
* **Design System:** `Syne` and `DM Mono` Typography
* **Animations:** CSS Transitions & JavaScript Intersection Observers

2. **Launch:**
